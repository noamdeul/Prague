# סוד הכתר האבוד של פראג

אתר סטטי פשוט למשחק משפחתי בסגנון חפש את המטמון, מותאם ליום טיול בפראג ולשימוש ישירות מהטלפון.

## מה יש בפרויקט
- `index.html` - מבנה הדף
- `style.css` - עיצוב RTL רספונסיבי ונוח למובייל
- `script.js` - כל הלוגיקה של המשחק, תחנות, ניקוד ושמירה ב־localStorage
- `manifest.json` - קובץ בסיסי לשיפור חוויית המובייל

## איך מעלים ל־GitHub Pages
1. צור ריפו חדש ב־GitHub.
2. העלה אליו את כל הקבצים בתיקייה הזאת.
3. היכנס ל־Settings של הריפו.
4. פתח את Pages.
5. תחת Build and deployment בחר:
   - Source: Deploy from a branch
   - Branch: `main` או `master`
   - Folder: `/root`
6. שמור.
7. אחרי דקה בערך האתר יעלה בכתובת ש־GitHub יציג.

## איך לערוך תחנות או ניקוד
כל התוכן נמצא בקובץ `script.js`.
- התחנות נמצאות במערך `stations`
- משימות הגיבוי נמצאות במערך `backupMissions`
- משימות הצילום נמצאות במערך `photoMissions`
- טיפים נמצאים במערך `tips`

אם רוצים לשנות ניקוד:
- בפונקציה `completeStation`
- בפונקציה `awardBonus`
- בכפתורי משימות הגיבוי בתוך `renderCardList`

## איך לאפס
בתוך האתר יש כפתור "איפוס משחק".
הוא מוחק את ההתקדמות מה־localStorage.

## איך לשנות עיצוב
רוב הצבעים והגדרות העיצוב נמצאים בתחילת `style.css` בתוך `:root`.
אפשר לשנות שם צבעים, גבולות ורדיוסים בקלות.
